[中文]LOC Calculation

Lucent Sky 支援 cloc 和 tokei 兩種程式碼行數計算工具來計算您的應用程式大小。

請參照下方的方法來安裝cloc 或 tokei :

【Windows】

點此下載 cloc 或 tokei

將下載後的程式放至您預計使用的路徑位置，使用該程式時需指定它的完整路徑。

您亦可透過下方的 Powershell 指令將該路徑目錄添加至 PATH 的環境變數中:

# "C:\Program Files\cloc" 這是範例的cloc安裝目錄，請確認您使用的是正確的路徑。
# 同樣的指令亦可使用於tokei，您只需要替換為正確的安裝路徑及檔名。

[System.Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\Program Files\cloc", [System.EnvironmentVariableTarget]::User)

以 administrator 身分執行 PowerShell 可以確保完整設置該機器系統上的 PATH 變數，若以當前user身分開啟 PowerShell 僅會對現行用戶的 PATH 變數做添加。

計算工具軟體安裝完後，您可透過直接執行該二進位檔案或完整路徑的指令來啟用，

如檔名: cloc-1.92 , 安裝路徑為:  C:\Program Files\cloc

# 使用完整路徑呼叫行數計算工具功能
C:\Program Files\cloc\cloc-1.92.exe '掃描專案路徑'

# 若您已經加入安裝路徑至 PATH 變數。
cloc-1.92.exe '掃描專案路徑'

透過指令驗證您的安裝是否正常: cloc --version 或 tokei --version

結果應會顯示已安裝的軟體版本，若有錯誤訊息則表示未正確安裝完成。

【MacOS】

您需要使用 Homebrew 這個套件管理工具來安裝 cloc 或 tokei，若您未安裝Homebrew，可於terminal使用以下指令來進行安裝: 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安裝好Homebrew後，執行指令以安裝 cloc 或 tokei: 

 brew install cloc 或 brew install tokei

Homebrew 會將該程式添加至您的 PATH 環境變數中，故您可直接使用 cloc 或 tokei 的指令，不用指定路徑。

透過指令驗證您的安裝是否正常: cloc --version 或 tokei --version

結果應會顯示已安裝的軟體版本，若有錯誤訊息則表示未正確安裝完成。

【Linux】

使用您的套件管理工具來進行安裝 cloc 或 tokei:

  • 對於 Debian-based distributions (如: Ubuntu)，使用 apt:  sudo apt install cloc 或 sudo apt install tokei


  • 對於 Red Hat-based distributions (如: CentOS, Fedora), 使用 yum 或 dnf :
    sudo yum install cloc 或 sudo yum install tokei;
    sudo dnf install cloc 或 sudo dnf install tokei。

套件管理工具會將該程式添加至您的 PATH 環境變數中，故您可直接使用 cloc 或 tokei 的指令，不用指定路徑。

透過指令驗證您的安裝是否正常: cloc --version 或 tokei --version

結果應會顯示已安裝的軟體版本，若有錯誤訊息則表示未正確安裝完成。

【如何透過 cloc 或tokei 進行掃描】

為了使用 cloc 或 tokei 計算出您的專案程式碼行數，您需要將程式專案的路徑作為執行 cloc 或 tokei 指令的第一個參數。

Example

如果您的程式放於 C:\MyProject 資料夾，您應操作如下:

  •  cloc: cloc C:\MyProject
  •  tokei: tokei C:\MyProject

注意事項:
- 一次僅計算單獨一個專案的程式碼行數: 請確保行數計算時，不得將不同的專案放入同個資料夾合併計算，因掃描時亦需分開進行掃描，故對應的程式碼行數也需分別計算。

舉例來說，若您有兩個不同的 .NET 專案和 一個Java 專案，您需要計算三次程式碼行數，而非一次，每個專案的行數應獨立計算以得到各別精準的結果。
